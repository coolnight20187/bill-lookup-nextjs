# Project Summary
The Bill Lookup System is a modern application designed to streamline billing processes for businesses by efficiently managing employee and customer data. Utilizing Supabase as its backend, the system ensures robust security through role-based access control and offers real-time capabilities, audit trails, and user management. This solution simplifies billing transactions, making it an essential tool for businesses looking to enhance their operational efficiency.

# Project Module Description
- **Database Schema (supabase_schema.sql)**: Defines the database structure for employees, members, warehouse, transaction history, work notes, audit logs, and system settings.
- **Row Level Security Policies (supabase_rls_policies.sql)**: Implements security policies to control data access based on user roles.
- **Authentication Setup (supabase_auth_setup.sql)**: Configures user authentication, including role management and JWT token customization.
- **Dashboard Management**: User interface for managing employees, customers, and inventory with role-based access control and responsive design.
- **Bill Lookup**: API integration for querying bills from multiple sources.
- **Warehouse Management**: Functionality for importing and exporting bills with tracking.
- **Employee Management**: Admin-only features for managing employee data and roles.
- **Customer Management**: Features for managing customer details and sales transactions.

# Usage
1. **Install Dependencies**: Ensure Node.js and pnpm are installed.
2. **Clone the Repository**: Clone the project repository to your local machine.
3. **Install Packages**: Run the command to install necessary packages.
4. **Setup Supabase**: Run the SQL scripts to set up the database schema and authentication.
5. **Configure Environment**: Set up environment variables for your Next.js application.
6. **Build the Project**: Execute the build command for your application.
7. **Run the Application**: Start the application using the appropriate command.
