# Project Summary
The Bill Lookup System is an advanced application aimed at optimizing billing processes for businesses by managing employee and customer data efficiently. It leverages Supabase as the backend, ensuring robust security through role-based access control, real-time capabilities, and comprehensive user management. The system enhances operational efficiency by simplifying billing transactions, making it an indispensable tool for businesses.

# Project Module Description
- **Database Schema (supabase_schema.sql)**: Defines the structure for employees, members, warehouse, transaction history, work notes, audit logs, and system settings.
- **Row Level Security Policies (supabase_rls_policies.sql)**: Implements security policies for data access control based on user roles.
- **Authentication Setup (supabase_auth_setup.sql)**: Configures user authentication, including role management and JWT token customization.
- **Dashboard Management**: User interface for managing employees, customers, and inventory with responsive design and role-based access control.
- **Bill Lookup**: API integration for querying bills from multiple sources.
- **Warehouse Management**: Functionality for importing and exporting bills with tracking.
- **Employee Management**: Admin features for managing employee data and roles.
- **Customer Management**: Features for managing customer details and sales transactions.
- **Dynamic API Routes**: New API routes for fetching bill and electricity data.

# Usage
1. **Install Dependencies**: Ensure Node.js and pnpm are installed.
2. **Clone the Repository**: Clone the project repository to your local machine.
3. **Install Packages**: Run the command to install necessary packages.
4. **Setup Supabase**: Run the SQL scripts to set up the database schema and authentication.
5. **Configure Environment**: Create a `.env.local` file with the necessary environment variables.
6. **Build the Project**: Execute the build command for your application.
7. **Run the Application**: Start the application using the appropriate command.
