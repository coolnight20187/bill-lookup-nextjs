# Project Summary
The Bill Lookup System is a sophisticated application aimed at enhancing billing processes for businesses by managing employee and customer data effectively. It leverages Supabase for backend functionalities, ensuring robust security through role-based access control and real-time capabilities. The application simplifies billing transactions, significantly improving operational efficiency and serving as a vital tool for enterprises.

# Project Module Description
- **Database Schema (supabase_schema.sql)**: Defines the structure for managing employees, members, warehouse data, transaction history, work notes, audit logs, and system settings.
- **Row Level Security Policies (supabase_rls_policies.sql)**: Implements security policies for data access based on user roles.
- **Authentication Setup (supabase_auth_setup.sql)**: Sets up user authentication, including role management and JWT token customization.
- **Dashboard Management**: User interface for managing employees, customers, and inventory with a responsive design and role-based access control.
- **Bill Lookup**: API integration for querying bills from various sources.
- **Warehouse Management**: Features for importing and exporting bills with tracking capabilities.
- **Employee Management**: Admin functionalities for managing employee data and roles.
- **Customer Management**: Tools for managing customer details and sales transactions.
- **Dynamic API Routes**: Updated API routes for fetching bill and electricity data with improved configurations.

# Usage
1. **Install Dependencies**: Ensure Node.js and pnpm are installed.
2. **Clone the Repository**: Clone the project repository to your local machine.
3. **Install Packages**: Run the command to install necessary packages.
4. **Setup Supabase**: Execute the SQL scripts to set up the database schema and authentication.
5. **Configure Environment**: Create a `.env.local` file with the necessary environment variables.
6. **Build the Project**: Execute the build command for your application.
7. **Run the Application**: Start the application using the appropriate command.
