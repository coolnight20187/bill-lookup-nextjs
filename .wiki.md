# Project Summary
The project aims to develop a modernized Bill Lookup System using Supabase as the backend. This system will enhance the management of employee and customer data, streamline transactions, and ensure robust security through role-based access control. It integrates various functionalities such as real-time capabilities, audit trails, and user management to provide a comprehensive solution for businesses managing billing processes.

# Project Module Description
- **Database Schema (supabase_schema.sql)**: Defines the structure of the database, including tables for employees, members, warehouse, transaction history, work notes, audit logs, and system settings.
- **Row Level Security Policies (supabase_rls_policies.sql)**: Implements security policies to control access to data based on user roles.
- **Authentication Setup (supabase_auth_setup.sql)**: Configures user authentication, including role management and JWT token customization.

# Usage
1. **Install Dependencies**: Ensure you have Node.js and npm installed.
2. **Clone the Repository**: Clone the project repository to your local machine.
3. **Install Packages**: Run the command to install necessary packages.
4. **Setup Supabase**: Run the SQL scripts to set up the database schema and authentication.
5. **Configure Environment**: Set up environment variables for your Next.js application.
6. **Build the Project**: Execute the build command for your application.
7. **Run the Application**: Start the application using the appropriate command.
